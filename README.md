# Tenda Mar 🏖️

Uma landing page premium e minimalista desenvolvida para a marca **Tenda Mar** — fabricante de tendas de praia leves, portáteis e criativas. O projeto tem como foco proporcionar uma experiência digital de **alto luxo** e imersiva para o usuário.

## 🛠️ Tecnologias Utilizadas

*   **HTML5 Semântico**
*   **CSS3 Vanilla** (Variáveis CSS, Flexbox, Grid, Media Queries, Keyframes Animations)
*   **JavaScript (Vanilla)** (Manipulação de DOM, Intersection Observer, Event Listeners)

## 🚀 Como Executar

Por ser um projeto puramente estático (sem framework complexo), não há necessidade de build.
1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` diretamente em seu navegador web.
