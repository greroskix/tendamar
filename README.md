# Tenda Mar 🏖️

Uma landing page premium e minimalista desenvolvida para a marca **Tenda Mar** — fabricante de tendas de praia leves, portáteis e criativas. O projeto tem como foco proporcionar uma experiência digital de **alto luxo** e imersiva para o usuário.

## ✨ Características do Projeto

*   **Design Luxury & Glassmorphism**: Interface moderna utilizando o efeito de vidro fosco (`backdrop-filter`) em cartões e elementos, proporcionando leveza visual.
*   **Hero com Parallax e Vídeo Background**: Um vídeo de fundo imersivo rodando em loop, com transições refinadas de cores e mesclagem de tipografia (`mix-blend-mode`).
*   **Tipografia Premium**: Utilização das fontes *Playfair Display* para títulos e *Inter* para o corpo de texto.
*   **Animações Suaves no Scroll**: O uso de `IntersectionObserver` em JavaScript nativo garante que os elementos surjam graciosamente (efeito *fade-in* e *slide-up*) conforme o usuário rola a página.
*   **Navegação Smooth**: Header flutuante que adquire o efeito *blur* ao rolar a página, contendo âncoras para rolagem suave (`scroll-behavior: smooth`) até os tutoriais e rodapé.
*   **Embed Nativo do TikTok**: Integração do player do TikTok preservando o layout responsivo (`aspect-ratio: 9/16`), perfeito para a exibição do tutorial de montagem.
*   **100% Responsivo (Mobile First)**: A estrutura utiliza *CSS Grid* e *Flexbox* para garantir uma adaptação perfeita desde monitores ultrawide até telas de celulares pequenos.

## 🛠️ Tecnologias Utilizadas

*   **HTML5 Semântico**
*   **CSS3 Vanilla** (Variáveis CSS, Flexbox, Grid, Media Queries, Keyframes Animations)
*   **JavaScript (Vanilla)** (Manipulação de DOM, Intersection Observer, Event Listeners)

## 🚀 Como Executar

Por ser um projeto puramente estático (sem framework complexo), não há necessidade de build.
1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` diretamente em seu navegador web.
